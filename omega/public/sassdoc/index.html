    button

    @mixin button($text--transform: $button--text--transform, $text--variant: $button--text--variant, $bg-color: $button--bg-color, $bg-color--hover: $button--bg-color--hover, $font-color: $button--font-color, $font-color--hover: $button--font-color--hover, $font--size: $button--font--size, $font--weight: $button--font--weight, $border: $button--border, $border-radius: $button--border-radius, $padding: $button--padding, $line-height: $button--line-height) { ... }
    View source

    Description

    Mixin for styling form button elements. Primarily for items like <a> <span> <button> <input type="submit">.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $text--transform

    CSS value for text-transform property.

    		String$button--text--transform
    $text--variant

    CSS value for font-variant property.

    		String$button--text--variant
    $bg-color

    CSS value for background color of the button.

    		String$button--bg-color
    $bg-color--hover

    CSS value for background color of the button during hover action.

    		String$button--bg-color--hover
    $font-color

    CSS value for font color of the button.

    		String$button--font-color
    $font-color--hover

    CSS value for font color of the button during hover action.

    		String$button--font-color--hover
    $font--size

    CSS value for font size of the button.

    		String$button--font--size
    $font--weight

    CSS value for font weight of the button.

    		String$button--font--weight
    $border

    CSS value for font-variant property.

    		String$button--border
    $border-radius

    CSS value for border radius value.

    		String$button--border-radius
    $padding

    CSS value for padding of the button element.

    		String$button--padding
    $line-height

    CSS value for line height.

    		String$button--line-height

    Example

    // This example doesn't pass any parameters, so takes on the appearance of the default values applied
// from the variables defined for default button values.
.my-default-button {
  @include button();
}

// This example passes some parameters to customize the appearance of the button.
.my-customized-button {
  @include button(
    $bg-color: #FF0000,
    $font-size: 32px,
    $text--variant: small-caps,
  );
}
